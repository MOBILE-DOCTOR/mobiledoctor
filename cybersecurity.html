Cybersecurity Hub

Welcome to the Cybersecurity Hub. Here you can learn about protecting yourself from digital threats and acquire new technical skills.

Common Cybersecurity Threats

Malware: Malicious software designed to disrupt operations or gain unauthorized access. This includes viruses, spyware, and ransomware.

Phishing: Fraudulent attempts, usually via email, to trick individuals into revealing sensitive information like passwords or credit card numbers.

Man-in-the-Middle (MitM) Attack: Attackers secretly intercept and relay communication between two parties to steal information. Public Wi-Fi is a common place for these attacks.

Cybersecurity Best Practices

  • Use Strong, Unique Passwords: Combine letters, numbers, and symbols. Use a different password for every account.
  • Enable Two-Factor Authentication (2FA): Adds a second layer of security, requiring a code from your phone in addition to your password.
  • Keep Software Updated: Updates often contain critical security patches to protect against new vulnerabilities.
  • Be Wary of Phishing: Always check the sender's email address and be suspicious of unexpected attachments or links.

Learn How to Install Linux in VirtualBox

Using a Virtual Machine (VM) is a safe way to test software or learn about new operating systems without affecting your main computer. Follow these steps to install Ubuntu, a popular Linux distribution, in VirtualBox.

Step 1: Download Required Software

  • VirtualBox: Download it from the official VirtualBox website. Install it on your computer.
  • Ubuntu ISO: Download the latest Ubuntu Desktop image (.iso file) from the official Ubuntu website.

Step 2: Create a New Virtual Machine

  1. Open VirtualBox and click the "New" button.
  2. Name and Operating System: Give your VM a name (e.g., "Ubuntu VM"). VirtualBox should automatically detect the Type and Version as Linux/Ubuntu.
  3. Memory Size: Allocate RAM for the VM. 4GB (4096 MB) is recommended for good performance.
  4. Hard Disk: Select "Create a virtual hard disk now". Choose "VDI (VirtualBox Disk Image)".
  5. Storage: Select "Dynamically allocated" for efficient disk space usage.
  6. File location and size: Allocate at least 25GB for the virtual hard disk. Click "Create".

Step 3: Install Ubuntu Linux

  1. Select your new VM from the list in VirtualBox and click "Start".
  2. A "Select start-up disk" window will appear. Click the folder icon and navigate to the Ubuntu .iso file you downloaded. Click "Start".
  3. The VM will boot into the Ubuntu installer. Click "Install Ubuntu".
  4. Follow the on-screen instructions for keyboard layout and updates.
  5. At "Installation type", choose "Erase disk and install Ubuntu". This is safe as it only affects the virtual disk, not your computer. Click "Install Now".
  6. Set your time zone, then create your user account with a username and password.
  7. The installation will proceed. Once it's finished, restart the VM as prompted.

Step 4: Install Guest Additions (Optional but Recommended)

After Ubuntu is running, go to the VM window's menu and select Devices > Insert Guest Additions CD image.... This improves performance and allows for features like a shared clipboard and resizable display. Run the installer that appears inside Ubuntu.
